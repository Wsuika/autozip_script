# Compress Folders Go Exercise

This program compresses all folders in the current directory into individual ZIP archives named after the corresponding folder.

```
    xx/currentFolder
            |- FolderA
            |- FolderB
            |- FolderC
            -- CompressToZip.exe
```

=>

```
    xx/currentFolder
            |- FolderA
            |- FolderB
            |- FolderC
            |- FolderA.zip
            |- FolderB.zip
            |- FolderC.zip
            -- CompressToZip.exe
```


## Usage
- git clone
- go build .
- copy the executive file to dest folder
- run the file
